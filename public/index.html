about

Build your own personal website in minutes

Github only provides a few boring templates for you to use with Github pages. What if you could get free awesome personal website templates that looked great to start and allowed you to make changes to them. The Personal Website lets you customize style and content, and then save all those changes to your Github repo. It's a free, simple, and easy CMS to build your personal brand!
