about

Build your own personal website in minutes

Github pages only supports a few boring templates. What if you could get free awesome personal website templates that looked great to start and allowed you to make changes to them later.

The Personal Website lets you customize style and content, and then save all those changes to this Github repository fork.

It's a free, simple, and easy CMS to build your personal brand!

Fork this repository to get started.
